# **FinTrack — Personal Finance App / Приложение для учёта финансов**

**FinTrack** — мобильное приложение для **учета доходов и расходов**, разработанное на **Flutter**.  
Позволяет **добавлять**, **редактировать** и **фильтровать транзакции**, хранить данные локально с помощью **Hive** и управлять состоянием через **Riverpod**.

## **Key Features / Основные возможности**

- **Add/Edit/Delete transactions / Добавление и редактирование транзакций**  
- **Filter by category/type / Фильтрация по категориям и типу**  
- **Local storage with Hive / Локальное хранение данных**  
- **Reactive UI with Riverpod / Реактивное обновление UI через Riverpod**  
- **Feature-based project structure / Feature-based структура проекта**  

## **Tech Stack / Технологии**

- **Flutter / Dart**  
- **Riverpod — state management**  
- **Hive — local storage**  
- **Intl — date & number formatting**  

## **Project Goals / Цели проекта**

- **Practice Flutter & Dart / Практика Flutter и Dart**  
- **Learn Riverpod and Hive on real project / Освоение Riverpod и Hive на реальном проекте**  
- **Portfolio-ready Flutter project / Проект для портфолио Flutter-разработчика**
